產品定位

App 是每日操作，Web 是長期價值。

如果 Web 只複製 App 的任務列表和分數，那會很弱。這裡改成提供大螢幕才適合看的資料、 家長與團體角色、活動營運工具，以及發表時能一眼理解系統的展示面。

資料流
  • App 完成任務產生分數、幣、健康與自律房資料
  • Web 整合分析轉成趨勢、風險、週報、星球成長
  • 角色互動家長、老師、團隊、營運者依權限查看
